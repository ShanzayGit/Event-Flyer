# 🎉  Event Flyer Page

A simple and responsive **Event Flyer webpage** created using **HTML and CSS**.  
This webpage promotes an exciting cultural festival by displaying event details, venue information, and an event description in a visually organized layout.


## 📌 Features

- Responsive page layout using viewport units
- Festival banner image
- Event title and headings with italic styling
- Highlighted date and time section
- Styled information cards using borders and box shadows
- Center-aligned content for clean presentation


## 🛠 Technologies Used

- HTML5
- CSS3

## 🚀 How to Run

1. Download or clone the project.
2. Open the folder in **VS Code**.
3. Open `index.html`.
4. Run using:
   - **Live Server Extension**, or
   - Open directly in browser.
